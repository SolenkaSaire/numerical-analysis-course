# MATLAB assignments in Numerical Analysis course!

Este repositorio contiene implementaciones en MATLAB de diversos métodos numéricos aplicados en el curso de Analisis de Métodos Numéricos. Cada archivo aborda un tema específico y proporciona una solución o aproximación numérica utilizando MATLAB.

## Contenido

### 1. Método de Bisección
- **Archivo:** `biseccion.m`
- **Descripción:** Implementación del método de bisección para encontrar raíces de funciones. El usuario proporciona la función, el intervalo y la tolerancia de error deseada.

### 2. Serie de Maclaurin para cos(x)
- **Archivo:** `cosenx.m`
- **Descripción:** Calcula la aproximación de la función cos(x) mediante la serie de Maclaurin. Permite al usuario ingresar el valor de x y el número de términos a evaluar.

### 3. Ajuste Lineal y Cuadrático con Error Cuadrático Medio
- **Archivo:** `errorCuadrado.m`
- **Descripción:** Realiza ajustes lineales y cuadráticos a un conjunto de datos y calcula los errores cuadráticos medios para cada ajuste. Indica cuál de los ajustes tiene un error menor.

### 4. Interpolación Polinómica de Lagrange
- **Archivo:** `lagrange.m`
- **Descripción:** Implementación de la interpolación polinómica de Lagrange para encontrar un polinomio que pase por un conjunto dado de puntos.

### 5. Método de Runge-Kutta de Cuarto Orden
- **Archivo:** `rungekutta4.m`
- **Descripción:** Resuelve una ecuación diferencial ordinaria utilizando el método de Runge-Kutta de cuarto orden. El usuario puede ingresar la condición inicial y el intervalo de evaluación.

**Nota:** Estos códigos se han desarrollado con fines educativos y demostrativos. Puedes utilizarlos como referencia para comprender los métodos numéricos y cómo implementarlos en MATLAB.
